# Deployment of Common OVA

The CommonOva is the core of the Uni5Xposure Platform used for Aggregating and all the findings in a single dashboard, along with providing features like Remediation, Report Generation and Launching Scans

## System Requirements
CPU: 8 Cores
Memory: 16 GB(Asset Count: 100)
Disk Space: 100GB
